Address Generator

Per questo compito si doveva costriure cona classe "AddressGenerator" che implementasse i metodi dell'interfaccia "IAddress" sotto rappresentata:
interface IAddress
{
string generateIPv4();
string generateSubnet();
}

Funzionamento
Per utilizzare la classe AddressGenerator, è sufficiente crearne un'istanza e chiamare i suoi metodi generateIPv4() e generateSubnet() per generare rispettivamente un indirizzo IPv4 dato dai 32 bit inseriti in fase di configurazione dell’oggetto e una maschera di sottorete calcolata tramite richiesta di CIDR da parte dell'utente.
